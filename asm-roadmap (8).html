; Cấp 4096 byte heap (phổ biến nhất)
mov rax, 9           ; sys_mmap
xor rdi, rdi         ; addr = NULL (kernel chọn)

mov rdx, 0x3         ; PROT_READ | PROT_WRITE

16-BIT MINI OPERATING SYSTEM

Một hệ điều hành được code hoàn chỉnh nhưng đơn giản — với đầy đủ tính năng để người học tự khám phá và sáng tạo. Viết bằng Assembly thuần túy, chạy trực tiếp trên BIOS real mode 16-bit.

⬡ TÍNH NĂNG HỆ THỐNG
🖥
Bootloader
Khởi động từ sector đầu tiên tại 0x7C00, load kernel vào bộ nhớ và trao quyền điều khiển.
⌨
Keyboard I/O
Đọc input từ bàn phím qua BIOS interrupt INT 16h. Xử lý từng ký tự, echo ra màn hình.
📺
VGA Text Mode
Ghi trực tiếp vào video memory 0xB8000. Hỗ trợ 16 màu, con trỏ, cuộn màn hình.
💾
Disk Read (INT 13h)
Đọc sector từ đĩa dùng BIOS INT 13h. Load thêm kernel module từ disk vào RAM.
🐚
Mini Shell
Shell đơn giản với các lệnh cơ bản: HELLO, CLEAR, REBOOT. Có thể mở rộng tùy ý.
⏰
System Clock
Đọc giờ/phút/giây từ BIOS INT 1Ah, hiển thị đồng hồ hệ thống dạng HH:MM:SS.
